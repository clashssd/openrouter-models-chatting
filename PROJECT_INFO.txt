PROJECT DESCRIPTION
===================

A comprehensive Python application for interacting with OpenRouter API,
providing model verification, adaptive fallback mechanisms, rate limiting,
caching, and interactive chat functionality with streaming support.

Designed for developers who need reliable access to multiple AI models
with automatic failover, rate limiting protection, and response caching.

KEY FEATURES
============

1. Multi-Model Management
   - Real-time model availability verification
   - Automatic fallback to alternative models
   - Model tier classification (S+, S, A+, A, B+, B, C)
   - Model metadata tracking and filtering

2. Rate Limiting and Retry Logic
   - Exponential backoff with jitter
   - Per-endpoint rate limit tracking
   - Configurable retry attempts (default: 5)
   - Thread-safe operations

3. Configuration and Profiles
   - Profile-based configuration management
   - System prompt customization
   - Settings inheritance and merging
   - YAML-based persistence

4. Chat Management
   - Multi-session support
   - Message history tracking
   - Session export (markdown, JSON, text)
   - Streaming response support

5. Response Optimization
   - LRU response caching
   - Token counting and cost estimation
   - Code syntax highlighting
   - Prompt templating system

6. Developer-Friendly CLI
   - Extensible command system
   - Interactive chat interface
   - Real-time model monitoring
   - Configuration management tools
